# 自我改进代理技能

这个技能旨在帮助我持续学习和改进，以更好地理解和满足你的需求。

## 功能概述

*   **偏好记忆**: 记录和回忆用户的个人偏好（例如，早晨天气预报，总结长度等）。
*   **行为调整**: 根据用户反馈自动调整我的响应和行为模式。
*   **指令学习**: 学习新的特定指令和上下文关联。
*   **优化操作**: 优化我在不同场景下的响应策略，例如在群聊中的参与度。

## 如何使用

当我需要记住您的偏好、调整我的行为或学习新的指令时，我会利用这个技能来记录和处理相关信息。您可以通过明确的指令来帮助我改进，例如：

*   “记住我喜欢...”
*   “下次请更...”
*   “当我提到‘XXX’时，请总是...”

## 内部资源

*   `references/preferences.md`: 存储用户的偏好设置。
*   `references/instructions.md`: 存储学习到的特定指令和上下文关联。
*   `scripts/learn_preference.py`: 用于解析和存储用户偏好。
*   `scripts/adjust_behavior.py`: 用于根据反馈调整行为。
*   `scripts/learn_instruction.py`: 用于解析和存储新指令。
